# RAG for AI-Generated Text Detection

A Retrieval-Augmented Generation (RAG) project for detecting AI-generated text using HC3, sentence embeddings, vector retrieval, and transformer-based models.

## Project Status

This repository is currently under development.

## Planned Features

- HC3 dataset exploration
- Text preprocessing
- Sentence embeddings
- Vector retrieval using FAISS
- RAG-based classification pipeline
- Evaluation using accuracy, precision, recall, F1-score, and confusion matrix
